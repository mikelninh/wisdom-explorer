# The Book of Wisdom — Interactive Explorer

A deep, balanced analysis of *"The Book of Wisdom"* by Revival of Wisdom. We examine each concept honestly: what's true, what's not, what believers say, and what science genuinely doesn't know yet.

**[View Live Site](https://mikelninh.github.io/wisdom-explorer/)**

## What's Inside

- **15 concepts analyzed** — from Christ Oil and Kundalini to Sacred Geometry and Astrotheology
- **Truth Score** — each concept rated on a 0-100% scale with a detailed justification for every score
- **Three perspectives per concept:**
  - **Our Analysis** — what science, history, and linguistics actually say
  - **The Believer's Case** — a fair, steel-manned version of how a true believer would respond
  - **What We Don't Know** — genuine open questions where science admits uncertainty
- **Custom SVG artwork** — 15 hand-crafted symbolic illustrations and an animated sacred geometry header
- **Sources & further reading** — peer-reviewed papers, historical texts, and philosophical works
- **Related concepts** — navigate connections between ideas
- **Filter by verdict** — Supported, Partially True, Unsupported, or Philosophical

## Our Approach

We try to be fair. The book mixes genuine esoteric traditions (Kabbalah, Hermeticism, Kundalini yoga) with unsupported claims. Rather than dismissing everything or accepting everything, we go concept by concept and separate what's real from what isn't — while admitting what we genuinely don't know.

Average truth score across all 15 concepts: **39%**

## Tech Stack

React + TypeScript + Vite, deployed on GitHub Pages via Actions.

## Run Locally

```bash
git clone https://github.com/mikelninh/wisdom-explorer.git
cd wisdom-explorer
npm install
npm run dev
```
